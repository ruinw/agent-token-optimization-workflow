# External Share Package

This is the external sharing package for the Agent Token Optimization Workflow repository.

目标：

- 用更少的文件呈现完整思路
- 降低阅读门槛
- 保留可执行性
- 方便发给团队外部成员或跨项目复用

## 目录说明

- `01-summary.md`
  一页式摘要，适合先看。

- `02-team-guide.md`
  团队级原则和规范。

- `03-sop.md`
  标准执行流程。

- `04-checklist.md`
  实施检查清单。

- `05-prompt-templates.md`
  可直接复用的 prompt 模板。

- `appendix-full-report.md`
  完整版详细报告，作为附录保留。

## 推荐阅读顺序

1. `01-summary.md`
2. `02-team-guide.md`
3. `03-sop.md`
4. `04-checklist.md`
5. `05-prompt-templates.md`
6. `appendix-full-report.md`
